## Objetivo
A proposta desse projeto é criar uma página web dinâmica utilizando formulários para enviar dados do front-end para o back-end.

A construção desse site permitirá o amadurecimento nas tecnologias e conceitos abaixo:
   - [X] Utilizar tags HTML básicas e semânticas;
   - [X] Utilizar tags de formulários;
   - [X] Utilizar Flexbox e media queries.
